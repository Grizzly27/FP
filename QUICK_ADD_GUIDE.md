### How to Use Quick Add

#### Step 1: Create a Recurring Task
First, you need at least one recurring task:
1. Go to **Tasks** tab
2. Click the **+** button
3. Fill in task details
4. Set **Recurrence** to Daily, Weekly, or Monthly
5. Save

#### Step 2: Navigate to Quick Add
1. In the **Tasks** view, click the **âš¡ Quick Add** tab
2. The liquid glass slider will smoothly animate to the new tab

#### Step 3: Select Your Options
1. **Choose a recurring task** from the dropdown
2. **Set date range**:
   - From: Start date (defaults to today)
   - To: End date (defaults to 7 days from today)
3. **Select family members**: Check the boxes for who should do the task
4. **Preview**: See how many tasks will be created

#### Step 4: Create Tasks
1. Review the preview showing:
   - Number of dates (based on recurrence pattern)
   - Number of selected members
   - Total tasks to be created
2. Click **âš¡ Create Tasks**
3. All tasks are created instantly!
4. You'll automatically switch back to the All Tasks tab

## ðŸ“Š Recurrence Patterns

### Daily
- Creates a task for **every day** in the date range
- Example: From 2/12 to 2/19 = 8 tasks per member

### Weekly
- Creates a task for the **same day of week** as the original task
- Example: If original was Monday, creates tasks for all Mondays in range

### Monthly
- Creates a task for the **same day of month** as the original task
- Example: If original was the 15th, creates tasks for the 15th of each month

## ðŸŽ¯ Tips

1. **Start with one recurring task** - Create the template first
2. **Use date range wisely** - Don't create too many tasks at once
3. **Check the preview** - Always verify before creating
4. **Mix and match members** - Select different members for different periods
5. **Individual instances** - Created tasks are individual (not recurring) so you can edit/delete them separately
